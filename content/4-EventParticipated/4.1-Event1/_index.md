# Summary Report: "Vietnam Cloud Day HCMC Connect Edition: GenAI & Data Track"

### Event Objectives

- Update on the latest technology trends in **Generative AI (GenAI)** and **Data Analytics**.
- Explore how to build a **Unified Data Foundation** to support AI applications.
- Learn from real-world solutions implemented by leading enterprises in applying GenAI to operations.
- Connect with experts and technology leaders to discuss digital transformation strategies.

### Speakers

- **Eric Yeo** - Country General Manager, AWS Vietnam
- **Trung Tri Nguyen** - Analytics Specialist Solutions Architect, AWS
- **Hailey Dinh** - Senior Sales Specialist (ASEAN Data & AI), AWS
- **Quang Chu** - Senior Solutions Architect, AWS Vietnam
- **Christopher Bennett** - CTO, TymeX

### Key Highlights

#### Unified Data Foundation

- The importance of breaking down data "silos" to create a **single source of truth**.
- Using services like **Amazon S3**, **AWS Glue**, and **AWS Lake Formation** to build a secure and scalable Data Lakehouse.
- Seamless data integration between Analytics and AI/ML to accelerate innovation.

#### Generative AI Applications with Amazon Bedrock

- How to simplify building and scaling GenAI applications using **Amazon Bedrock**.
- Leveraging leading Foundation Models (FMs) via a single API without managing complex infrastructure.
- Security and data privacy features when deploying GenAI for enterprises.

#### Digital Transformation in Finance (Use Case: TymeX)

- Practical sharing on building an **end-to-end personalized lending solution**.
- Applying advanced data analytics for risk assessment and optimizing customer experience.

### Key Takeaways

#### Modern Data Strategy

- **Data is fuel for AI**: For GenAI to work effectively, businesses need a solid data strategy ensuring quality, availability, and security compliance.
- **Zero-ETL**: The trend of direct integration between database services (like Amazon Aurora, DynamoDB) and data warehouses (Amazon Redshift) to minimize manual data movement.

#### The Power of Generative AI

- GenAI is not just a trend but is creating real impacts on the economy and labor productivity.
- Potential applications: Automated customer care, marketing content creation, coding assistants, and intelligent data analysis.

#### Serverless Architecture for Data & AI

- Using **Serverless** architecture helps reduce the operational burden, allowing teams to focus on developing business logic and AI models.

### Applying to Work

- **Review Data Architecture**: Re-evaluate the current data organization of the project; consider adopting the Data Lakehouse model.
- **Experiment with Amazon Bedrock**: Propose building small Proof of Concepts (POCs) using Bedrock to solve specific problems (e.g., text summarization, internal chatbots).
- **Optimize ETL Processes**: Research AWS Zero-ETL solutions to reduce data latency.
